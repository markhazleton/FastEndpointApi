What is FastEndpoints?

FastEndpoints is a lightweight REST API framework for ASP.NET that implements the REPR (Request-Endpoint-Response) Pattern. It's designed to be a developer-friendly alternative to Minimal APIs and MVC Controllers, significantly reducing boilerplate code while maintaining high performance.

Key Features

  • CRUD Operations: Complete set of endpoints for creating, reading, updating, and deleting Person entities
  • In-Memory Data Store: Simple in-memory collection for demo persistence
  • Service Layer: Clean abstraction of business logic
  • Smart Data Mapping: Built-in mapping between models and entities
  • Dependency Injection: Practical DI usage
  • Reusable Base Classes: Promotes code reuse
  • HATEOAS Links: Hypermedia links in API responses
  • Interactive Documentation: Swagger/OpenAPI integration

Getting Started

  1. Clone this repository:
    git clone https://github.com/MarkHazleton/FastEndpointDemo.git
  2. Build and run the project:
    cd FastEndpointApi
    dotnet build
    dotnet run
  3. Open Swagger UI to explore the API.

API Endpoints

Method Endpoint Description
POST /person Create a new person
GET /person/{id} Get a person by ID
GET /person Get all persons
PUT /person/{id} Update a person
DELETE /person/{id} Delete a person

Implementation Details

  • Data Seeding: 5 unique people seeded using Bogus
  • Global Error Handling: Consistent JSON error responses
  • HATEOAS: Hypermedia links in responses
  • Service Layer: In-memory CRUD logic

Why Use FastEndpoints?

  • High performance, minimal boilerplate
  • Auto-discovery of endpoints
  • Built-in validation and security

Glossary

  • REPR Pattern: Request-Endpoint-Response Pattern
  • DTO: Data Transfer Object
  • REST: Representational State Transfer
  • HATEOAS: Hypermedia as the Engine of Application State
  • Swagger/OpenAPI: API documentation and exploration tools
  • Endpoint: A specific URL for API access
  • Dependency Injection (DI): Supplying dependencies to objects
  • Mapper: Transforms data between types
  • Entity: Domain model class
  • Minimal API: Simplified HTTP APIs in ASP.NET Core
  • Fluent Validation: Validation library for .NET
  • NuGet Package: .NET package manager
  • ASP.NET Core: Modern, cross-platform web framework
